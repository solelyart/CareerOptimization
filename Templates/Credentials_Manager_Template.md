## Security & Password Best Practices

### Strong Password Guidelines

**Weak Password Examples (NEVER USE):**
- ‚ùå Password123
- ‚ùå Kristen2024
- ‚ùå 123456789
- ‚ùå qwerty
- ‚ùå YourName!

**Strong Password Examples:**
- ‚úÖ 8$mK9#pL2@vN5qR7 (random characters)
- ‚úÖ correct-horse-battery-staple-2024! (passphrase method)
- ‚úÖ BlueElephant$Dances!47 (memorable but complex)

**Password Generator Tools:**
- 1Password generator (built-in)
- LastPass generator
- Bitwarden generator
- Random.org/passwords

**Requirements for Strong Passwords:**
- At least 16 characters
- Mix of uppercase, lowercase, numbers, symbols
- No dictionary words
- No personal info (name, birthday, pet name)
- Unique for every platform

---

### Two-Factor Authentication (2FA) Setup

**Recommended 2FA Method: Authenticator App**

**Best Authenticator Apps:**
- **Authy** (Free) - Multi-device sync, cloud backup
- **Google Authenticator** (Free) - Simple, widely supported
- **Microsoft Authenticator** (Free) - Works with Microsoft accounts
- **1Password** (Paid) - Built-in authenticator if you use 1Password

**How to Set Up 2FA:**
1. Go to platform's Security Settings
2. Choose "Enable Two-Factor Authentication"
3. Select "Authenticator App" (not SMS if possible - SMS is less secure)
4. Scan QR code with authenticator app
5. **SAVE BACKUP CODES** (store in password manager or print and secure)
6. Test login with 2FA to ensure it works

**Enable 2FA on These Platforms (Priority Order):**
1. ‚úÖ Upwork (protects your earnings)
2. ‚úÖ LinkedIn (prevents account takeover)
3. ‚úÖ Email account (protects everything)
4. ‚úÖ Banking/PayPal (financial security)
5. ‚úÖ All other platforms (as available)

## Credentials Sharing with Contractor (If Hiring One)

**If you hire a contractor to set up platforms, here's how to share credentials securely:**

### Option 1: Password Manager Guest Access
- 1Password: Create "Guest" account, share specific credentials
- Bitwarden: Create "Organization," invite contractor with limited permissions
- LastPass: Use "Share" feature for specific items
- **Revoke access after project completion**

### Option 2: Create Temporary Credentials
- Create accounts with temporary passwords
- Share via secure methods (see below)
- Change passwords after contractor delivers work

### Option 3: Contractor Creates Accounts (You Don't Share Passwords)
- Give contractor YOUR information (resume, bio, etc.)
- Contractor creates accounts using YOUR email
- You set passwords via "forgot password" flow
- More secure but requires more coordination

**NEVER:**
- ‚ùå Email passwords in plain text
- ‚ùå Share via unsecured document
- ‚ùå Give contractor access to financial accounts
- ‚ùå Leave contractor access enabled after project ends

**Secure Sharing Methods:**
- ‚úÖ 1Password/LastPass sharing feature
- ‚úÖ Encrypted file (7-Zip with password, sent separately)
- ‚úÖ One-time secret (onetimesecret.com) - link expires after viewing
- ‚úÖ Secure messaging (Signal, ProtonMail encrypted)

## Emergency: Account Compromised

**If you suspect an account has been hacked:**

1. **Immediately:**
   - Change password
   - Log out all devices
   - Enable 2FA (if not already)
   - Review recent activity/transactions

2. **Check for damage:**
   - Upwork/Freelance platforms: Check for unauthorized proposals/messages
   - LinkedIn: Check for spam posts from your account
   - Banking: Check for unauthorized transactions

3. **Report:**
   - Contact platform support
   - File report if financial fraud involved
   - Warn network connections (if account sent spam)

4. **Prevent future:**
   - Identify how breach occurred (weak password? phishing? malware?)
   - Install antivirus/anti-malware
   - Review all other accounts for similar vulnerabilities
