# VA Quality Assurance (QA) Checklists

This document provides QA checklists for the Virtual Assistant to ensure each platform profile is 100% complete.

## General Checklist (All Platforms)

- [ ] Profile picture uploaded (correct size)
- [ ] Professional bio/summary is complete and error-free
- [ ] All relevant skills from resume are added
- [ ] Work experience matches resume
- [ ] Education matches resume
- [ ] All links (portfolio, LinkedIn) are correct

## Tutoring Platform Checklist (Wyzant, Chegg)

- [ ] All tutoring subjects are listed with sub-topics
- [ ] Hourly rate is set correctly
- [ ] Target audience description is included

## Freelance Platform Checklist (Upwork, Braintrust)

- [ ] All 4 portfolio case studies are uploaded with descriptions
- [ ] All 3 core freelance services are listed with descriptions
- [ ] Hourly consulting rates (tiered) are set correctly

## Course Platform Checklist (Teachable, Udemy)

- [ ] Course title and subtitle are correct
- [ ] Course outline is added
- [ ] Course pricing (launch and standard) is set correctly

## Job Board Checklist (Indeed, FlexJobs)

- [ ] Enhanced resume is uploaded and parsed correctly
- [ ] Job alerts are created for all target titles and keywords
- [ ] Work arrangement and salary preferences are set
