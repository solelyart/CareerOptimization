# Time Etc + Zapier Integration Guide

This document outlines how to connect Time Etc with other apps via Zapier to automate your virtual assistant workflow.

## Available Triggers (Events that start a workflow)

| Trigger | Description |
|---------|-------------|
| **Completed Task** | Triggers when a task has been marked as complete by your assistant |
| **New Task** | Triggers when a new task is created by your assistant |
| **New Comment** | Triggers when a new comment is posted by your assistant |
| **Time Logged to Task** | Triggers when time is logged to a task by your assistant |

## Available Actions (Events a Zap performs)

| Action | Required Fields |
|--------|-----------------|
| **Create Task** | Title, Description, Assign to |

## Popular Integration Templates

### Task Management Integrations

| Integration | Description |
|-------------|-------------|
| Gmail → Time Etc | Create Time etc tasks for new Gmail emails |
| Google Sheets → Time Etc | Create tasks in Time etc with new rows added or updated |
| Google Drive → Time Etc | Create Time etc tasks from new files in folders |
| Asana → Time Etc | Generate tasks in Time etc with new tags added to Asana tasks |
| HubSpot → Time Etc | Create Time etc tasks for new HubSpot contacts |
| Basecamp → Time Etc | Create Time etc tasks for new to-dos in Basecamp |
| Trello → Time Etc | Create tasks from new labels added to Trello cards |

### Notification Integrations

| Integration | Description |
|-------------|-------------|
| Time Etc → Slack | Send notifications for new comments or tasks |
| Time Etc → Microsoft Teams | Send messages for new tasks |
| Time Etc → Google Sheets | Create rows for new time logged |

## How to Set Up the Integration

1. Log into your Zapier account at https://zapier.com
2. Click "Create Zap" or search for "Time Etc" in the app directory
3. Select Time Etc as your trigger or action app
4. Connect your Time Etc account when prompted (you'll need to authorize Zapier)
5. Configure your trigger/action settings
6. Test and activate your Zap
