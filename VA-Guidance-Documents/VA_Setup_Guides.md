# VA Step-by-Step Setup Guides

This document provides platform-specific setup guides for the Virtual Assistant.

## General Instructions

- Use the `Kristen.Blanks.Platforms@gmail.com` email for all sign-ups.
- Use the temporary master password for all accounts.
- Use the "Warm/Approachable" headshot for all platforms unless otherwise specified.
- Refer to the `Platform_Strategy.md` document for rates, services, and other strategic decisions.

## Wyzant & Chegg (Tutoring)

1. **Create Account:** Sign up as a tutor.
2. **Complete Profile:** Fill out all profile sections using the `Core_Content.md` document.
3. **Set Subjects & Rates:** Use the subjects and rates from `Platform_Strategy.md`.
4. **Submit Application:** Submit the profile for review.

## Upwork & Braintrust (Freelance)

1. **Create Account:** Sign up as a freelancer.
2. **Complete Profile:** Fill out all profile sections, including skills and experience.
3. **Add Portfolio:** Add the 4 case studies from `Portfolio_Case_Studies.md`.
4. **Set Services & Rates:** Use the services and rates from `Platform_Strategy.md`.

## Teachable & Udemy (Courses)

1. **Create Account:** Sign up as an instructor.
2. **Create Course Shell:** Create a new course with the title and subtitle from `Platform_Strategy.md`.
3. **Add Course Outline:** Add the high-level course outline.
4. **Set Pricing:** Set the launch and standard pricing.

## Indeed, FlexJobs, HigherEdJobs (Job Boards)

1. **Create Account:** Sign up as a job seeker.
2. **Upload Resume:** Upload the `Kristen_Blanks_Resume_Enhanced.pdf`.
3. **Set Job Alerts:** Create job alerts using the keywords and preferences from `Job_Board_Strategy.md`.

## BELAY & Toptal (Hiring Processes)

1. **Start Application:** Begin the application process.
2. **Provide Information:** Fill out all required information from the `Core_Content.md` and resume.
3. **Submit Application:** Submit the application for review.
