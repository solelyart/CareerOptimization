## 6. Common Issues & Solutions

### Issue: Resume Parsing Errors on Indeed/FlexJobs

**Solution:**
- Don't rely on auto-parse
- Manually review every field after upload
- Fix dates, titles, companies manually
- Simplify formatting if parser struggles (remove tables, columns)

---

### Issue: Platform Requires Phone Verification

**Solution:**
- Contact client for real-time verification code
- Set up dedicated time for platforms requiring SMS/call verification
- Use client's phone: (901) 277-5597

---

### Issue: Platform Asks for Certifications Client Doesn't Have Yet

**Solution:**
- Skip certification section or leave blank
- Note in deliverables: "Client is currently pursuing: Salesforce Admin, CSM, ECBA"
- Explain client can update once certifications are earned

---

### Issue: Can't Create Account - Email Already in Use

**Solution:**
- Client may already have an account
- Contact client to confirm
- If account exists, request login credentials
- Optimize existing profile instead of creating new

---

### Issue: Platform Has Mandatory Video Introduction

**Solution:**
- Note in deliverables that video is optional or requires client to record
- Provide script for client to use if they choose to record
- Don't attempt to record video as contractor (must be client)

---

### Issue: Platform Approval Takes Weeks (Toptal, Chegg, BELAY)

**Solution:**
- This is expected and normal
- Complete application to highest standard
- Note in deliverables: "Application submitted [date], awaiting [Platform] approval (typical timeline: 1-4 weeks)"
- Contractor responsibility ends at application submission
- Platform approval/rejection is outside your control

---

### Issue: Client Feedback Requires Significant Content Changes

**Solution:**
- 2 rounds of revisions are included in SOW
- Make requested changes within 2 business days
- If client requests go beyond 2 rounds or outside original scope, discuss additional fees
- Document all changes requested and completed

---

### Issue: Platform Changes Design or Features Mid-Project

**Solution:**
- Adapt to new interface
- Update your instructions for future reference
- Note changes in final deliverables report
- No need to notify client unless it significantly impacts deliverable

---

### Issue: Running Behind Schedule

**Solution:**
- Communicate early with client (don't wait until deadline)
- Explain specific reason for delay
- Provide revised timeline
- Work extra hours to catch up if possible
- Be transparent and professional
