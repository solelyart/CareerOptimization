# Teachable School and Course Shell Setup (Priority: MEDIUM)

**Objective:** Create a Teachable school and set up a course shell.

**Required Elements:**

*   **School Creation:** Create and name the school: "Kristen Blanks Tech Academy" (or the client's preferred name).
*   **School Branding:** Configure the school's branding with a logo and colors (use client-provided assets or simple defaults).
*   **Instructor Profile:** Complete the instructor profile.
*   **Course Shell:** Create a course shell with:
    *   Course title
    *   Landing page draft
    *   Curriculum outline (5-8 sections)
    *   Pricing structure
*   **Payment Processor:** Document the setup instructions (the client will connect their own payment method).

**Success Criteria:**

*   School is live.
*   Course is saved as a draft and ready for content.
