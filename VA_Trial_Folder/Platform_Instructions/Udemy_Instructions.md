# Udemy Instructor Account and Course Shell Setup (Priority: MEDIUM)

**Objective:** Create an instructor account on Udemy and set up a course shell.

**Required Elements:**

*   **Instructor Account:** Create and get the instructor account approved.
*   **Instructor Profile:** Complete the instructor profile.
*   **Bio:** Write a 200-300 word bio that is student-friendly.
*   **Course Shell:** Create a course shell with the following:
    *   **Course Title:** "Complete Business Systems Analyst Career Path" (or similar).
    *   **Course Description:** 200-300 words.
    *   **Course Outline:** 5-8 sections with 3-5 lectures each (outline only, not full content).
    *   **Target Audience:** Define the target audience.
    *   **Learning Objectives:** 5-8 specific outcomes.
    *   **Pricing Tier:** Select a pricing tier.

**Note:** The client will create the actual course content after the profile is set up.

**Success Criteria:**

*   Instructor account is approved.
*   Course shell is saved as a draft.
