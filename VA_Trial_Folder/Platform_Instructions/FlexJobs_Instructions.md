# FlexJobs Profile Setup (Priority: MEDIUM)

**Objective:** Create a complete FlexJobs profile to receive relevant remote job alerts.

**Required Elements:**

*   **Profile Completion:** Complete all sections of the profile.
*   **Resume Upload:** Upload the latest version of the resume.
*   **Work Style Preferences:** Set to "Remote" as the primary preference.
*   **Job Alerts:** Configure 3-5 job alerts.
*   **Job Categories:** Select "Business Analysis," "Salesforce," and "Project Management."
*   **Skills & Interests:** Complete the "Skills & Interests" section.

**Client Responsibility:** The client will purchase the FlexJobs subscription (~$25/month).

**Success Criteria:**

*   Profile is complete.
*   Receiving relevant job alerts.
