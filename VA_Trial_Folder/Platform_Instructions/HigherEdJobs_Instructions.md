# HigherEdJobs Profile Setup (Priority: LOW)

**Objective:** Create a comprehensive academic profile on HigherEdJobs to attract adjunct and faculty positions.

**Required Elements:**

*   **Full Academic Profile:** Complete all sections of the academic profile.
*   **CV Upload:** Upload the resume formatted as a CV.
*   **Education Section:** Include all degrees.
*   **Teaching Experience:** Highlight all teaching experience.
*   **Job Alerts:** Configure 3-5 job alerts for adjunct/faculty positions.
*   **Desired Locations:** Set to "Remote," "North Carolina," or "Open to relocation."

**Optimization Requirements:**

*   Use an academic tone and language.
*   Incorporate a teaching philosophy into the profile.
*   Include a section for research and publications, if applicable.

**Success Criteria:**

*   Profile is complete.
*   Job alerts are active.
